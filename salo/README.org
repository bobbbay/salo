This is the WIP implementation for Salo, SEMC's configuration toolset. A copy of the standard can be found in [[./standard]].

To open the REPL, run:

#+begin_src shell
$ cargo run
#+end_src

To evaluate a specific file, run:

#+begin_src shell
$ cargo run -- eval ./examples/simple.sa
#+end_src

To run the built-in tests, run:

#+begin_src 
$ cargo t  
#+end_src

To run benchmarks, run:

#+begin_src 
$ cargo bench  
#+end_src
